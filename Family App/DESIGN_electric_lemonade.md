# Electric Lemonade — Design System

Bold, graphic, and high-energy. Electric yellow backgrounds with deep purple accents and hard offset shadows. Activated by `data-theme="electric"` on `<html>`.

| Token | Value | Use |
|---|---|---|
| `--lavender` | `#7c3aed` | Primary accent — nav active, calendar headers, Stella |
| `--lavender-light` | `#ede9fe` | Violet hover tints |
| `--lavender-dark` | `#4c1d95` | Dark violet text/icons |
| `--mint` | `#16a34a` | Roman — electric green |
| `--mint-light` | `#dcfce7` | Green tints |
| `--mint-dark` | `#14532d` | Dark green text |
| `--yellow` | `#ea580c` | Layla — vivid orange (yellow avoided: bg is yellow) |
| `--yellow-light` | `#ffedd5` | Orange tints |
| `--yellow-dark` | `#9a3412` | Dark orange text |
| `--pink` | `#ec4899` | Mom — hot pink |
| `--pink-light` | `#fce7f3` | Pink tints |
| `--pink-dark` | `#9d174d` | Dark pink text |

## Member Colors
- 👩 Mom → Hot Pink `#ec4899`
- 👨 Dad → Electric Blue `#2563eb`
- 🌸 Stella → Vivid Violet `#7c3aed`
- 🦕 Roman → Electric Green `#16a34a`
- 🌻 Layla → Vivid Orange `#ea580c`

## Typography
- Display / Headlines: `'Epilogue'` — 700–900 weight, uppercase
- Body: `'Plus Jakarta Sans'` — 400/500 weight
- Labels: `'Spline Sans'` — 700 weight, uppercase, wide tracking

## Shape
- Card radius: `16px` — same structure, shadows make cards "pop"
- Hard shadows replace soft diffuse shadows throughout
- Borders: `2px solid #581c87` on interactive elements

## Reference
Designed as "Sunshine Hub / FamPulse — Electric Lemonade."
Color system based on Material You with yellow primary container and purple secondary.
