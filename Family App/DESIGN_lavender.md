# Lavender Garden — Design System

Soft, warm, and cozy. Built around lavender purples and gentle pastels that feel approachable for the whole family.

## Color Palette

| Token | Value | Use |
|---|---|---|
| `--lavender` | `#c9a8e0` | Primary accent — nav active, calendar headers |
| `--lavender-light` | `#ecdff7` | Hover backgrounds, section tints |
| `--lavender-dark` | `#9b7bbf` | Icons, headings, text on light |
| `--mint` | `#8ecf8e` | Secondary accent |
| `--pink` | `#f4a0b5` | Mom |
| `--blue` | `#87c4e0` | Dad |
| `--yellow` | `#f5d87a` | Layla |
| `--peach` | `#f5c09a` | Shared / unassigned events |
| `--bg` | `#fdf6ff` | Page background |
| `--text` | `#3d2c4e` | Body text |
| `--text-muted` | `#8a7898` | Labels, secondary text |
| `--border` | `#e8d5f5` | Card and input borders |
| `--shadow` | `0 4px 20px rgba(180,140,220,0.12)` | Soft diffuse shadow |

## Typography
- Font: `'Segoe UI', system-ui, -apple-system, sans-serif`
- Base: 15px / 1.5 line-height
- Headings: 600 weight, sentence case

## Shape
- Card radius: `16px` — Pill radius: `9999px`
- Input radius: `8px`
